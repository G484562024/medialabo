各授業の学習事項

第1回　WWWとHTML

学習内容はHTMLをあつかう上での様々な前提知識を学びました。URLの構造やHTMLの記述方法、

タグを活用しやページの装飾などです。URLはプロトコル、サード、パスの順に構成されています。

どれも大事なものでプロトコルは約束事、サーバはwebページの配布元、パスはファイルのサーバ

における配置場所です。

HTMLは要素で作られます。これはあらかじめ決められた多種多様なものがあります。MDN Web Docs

使用して作られています。タグを使用すると幅広くページを彩れます。

演習の成果:演習1-4:HTMLとさまざまな要素

第2回　CSSによるWebページの装飾

第２回はcssについてです。cssとはhemlで作成したWEBページを装飾するものです。

構文を書き、そこにカラーをすることでページに彩りを加え可読性や誘目性などを高

めることができます。デザイン性を高めることでページ使用者に考慮したいわば遊び

心を込めたWebページを作成できます。また、cssではカラーコードを指定することで

より高度なデザインを作成できます。ただし、注意点として構文がHTML並みに多いこ

とです。Webなどで調べながら書く必要がある点も重要です。コンパイル方法も簡単に

Firefoxで行えます。そうすることで配色を簡単に確かめることができ完成への近道

になります。ただし、保存はされないのでカラーコードをツールで調べ反映させる必

要があります。
